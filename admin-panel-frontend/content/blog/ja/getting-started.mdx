# Appuri Hogosha の使い方

**Appuri Hogosha** は、教師と管理者が保護者と迅速かつ効率的にコミュニケーションを取るための学校通知プラットフォームです。

## できること

- 📢 保護者へのプッシュ通知の即時送信
- 📋 生徒・保護者情報の管理
- 👥 生徒のグループ分け
- 📱 モバイルアプリまたはSMSによる連絡

## ステップ1：管理パネルにログイン

ブラウザで管理パネルを開き、学校の管理者アカウントでログインしてください。アカウントをお持ちでない場合は、システム管理者にお問い合わせください。

## ステップ2：生徒と保護者を追加

サイドバーの**生徒**に移動します。**生徒を追加**をクリックし、生徒の名前、クラス、保護者の連絡先を入力します。各保護者には Appuri Hogosha モバイルアプリのインストール招待が届きます。

## ステップ3：グループを作成

グループを使用すると、ターゲットを絞った通知を送ることができます。**グループ**に移動し、クラスグループ（例：「5年A組」）を作成して生徒を割り当てます。

## ステップ4：最初の通知を送信

**メッセージ**に移動し、**新しいメッセージ**をクリックします。受信者グループまたは個別の保護者を選択し、メッセージを入力して**送信**をクリックします。保護者は数秒以内にスマートフォンにプッシュ通知を受け取ります。

## 効果的な通知のヒント

1. **簡潔に書く** — 保護者はスマートフォンでメッセージをすばやく読みます。
2. **グループを活用する** — 関係のないメッセージをすべての保護者に送るのは避けましょう。
3. **メッセージをスケジュール** — 計画的なお知らせには日時ピッカーをご利用ください。

## サポートが必要な場合

問題が発生した場合は、学校のシステム管理者にお問い合わせいただくか、アプリの提供状況を確認するために[保護者通知ページ](/parentnotification)をご覧ください。
